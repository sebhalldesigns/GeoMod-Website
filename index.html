Overview

GeoMod (the GEOmetric MODular application kernel) is an open-source, cross-platform framework for building modern design software. It is designed around a highly modular architecture that allows for a highly flexible, configurable and powerful system.

Example Applications

  • Game design & engine development
  • Game deployment
  • Domain-specific CAD software
  • Product visualisation and presentation
  • Integrating 3D graphics into existing systems
